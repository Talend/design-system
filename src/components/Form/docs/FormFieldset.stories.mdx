## Content

### Inline message

By default, the error message is displayed using the inline message component, it appears at the bottom of the field.
When a fieldset or a field is bigger than 15rem, the **inline message** (often error message) should appear on top instead of bottom.

In case of error after submission, the focus on the screen should be done on the fieldset or field errored. If more than one field in error, the focus should be placed on the first one.

## Interactions

### Conditional field

Conditional fields allows the user to access a fieldset only when conditions are met in a previous input. 

Conditional fields can appears following:
- checkboxes
- radio buttons
- switch
- select
