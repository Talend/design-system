## Style

### Density

In order to improve the content readibility we allow the user to change the list density. 

The **default** density is 5rem

	src="https://www.figma.com/file/YOFfHsT7Gf3FUUm74i4thw/List?node-id=691%3A75433"

## Interaction

### Selection

The user can select one or multiple rows by clicking on the checkbox. It's also possible to select all the items by clicking on the top checkbox.

### Switching view

The user can switch the list density by opening the **Manage view** popover. The list density update is immediate.
