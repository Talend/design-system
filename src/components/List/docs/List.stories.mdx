## Interaction

### Sorting

Not all columns can be used for sorting, it should be optional to allow sorting by column.
If the column has multiple values in a cell, the content of the cells should be in alphabetical, numerical, or in date order depending on the content type unless there is a special order to use â€œstate/defined order".
If the content of the cell is integers, then the content should be ordered numerically.

### Resizing columns

When a column is resizable, it's visible while mouse hovering an header cells intersection.The cursor is changing when the user mouse over between two column.
- The resize must respect the min-width
- Not all the columns are resizable, for example there is no interest to enlarge a column with only one icon inside
- If columns are enlarged more than the screen container width, then an horizontal scrollbar is displayed to ensure a better navigation in the screen
