## Style

### Sizes

The max-width for a step is 15rem, if the label is longer display an ellipsis.

### Variations

**Description**

Having a step description is optional. The description offers additional information or guidance to the user.

**Sub step**

Sub steps are optional. They should be used to avoid having too many steps, or too long steps.
