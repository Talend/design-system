# Column chooser

The Column chooser gives the possibility to display or undisplay the column, mostly in the List and the Data Grid.

## Style

### Sizes 

The column chooser has a **54rem max-height**, corresponding to 8 entries.

### Variations

Not all the columns can be edit: a column can be **locked**.

## Interactions

### Filtering

The user can search for a specific column by using the search input. 

<FigmaImage
    src="https://www.figma.com/file/WYRKYbMA8gohDx7ShkYG4F/Column-chooser?node-id=51%3A3465"
	alt="filtering column chooser flow"
/>

### Validation 

- The user must clicks on the **Apply button** to validate his choice. 
- Clicking outside the Column chooser will cancel his choice. 

## Content 

### Long title

If the title is too long, display an ellipsis at the end. The full title is available on hover displayed in a tooltip.
