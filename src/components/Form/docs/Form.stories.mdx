## Style 

The form width is adaptative, it depends on the form container size. The form is horizontally **centered** in its container.

| Container         | Width      |
| ---------------- | ---------- |
| > 1536px         | max-width: 78rem |
| 1536px < 834px   | 55%        |
| < 834px          | 100%       |
