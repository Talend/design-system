## Content

### Text too long 

When the content is longer than the cell width, display an ellipsis at the end. On hover, the full text is displayed in a tooltip.

<FigmaImage
	src="https://www.figma.com/file/YOFfHsT7Gf3FUUm74i4thw/List?node-id=943%3A851"

/>

### Empty cell

If a cell is empty, use a dash (-)
